HTML TAGS PURPOSE
TABLE DECLARES A NEW TABLE
TR CREATE A NEW ROW
TD CREATE A NEW CELL CONTAINED WITHIN PARENT ROW
TH CREATE A TABLE HEADER CELL-BOLD CHARACTERS
THEAD DEFINE A GROUP OF A TABLE HEADER
TBODY DEFINE A GROUP OF A TABLE BODY
TFOOT DEFINE A GROUP OF A TABLE FOOTER
ATRIBUTES PURPOSE
COLSPAN='X" MAKES A TABLE'S CELL SPAN MORE THAN 1 COLUM
ROWSPAN="X" MAKES A TABLE'S CELL SPAN MORE THAN 2 ROWS
